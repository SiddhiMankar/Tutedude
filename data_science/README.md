# Data Science Assignments

This directory contains assignments and projects related to data science concepts implemented in Python.

## Projects

### Employee Management System (`assignment1.py`)
A command-line interface (CLI) application to manage employee records.
- **Features**:
  - Add new employees with ID, Name, Age, Department, and Salary.
  - Delete employees by ID.
  - Update existing employee information.
  - Search for employees by ID.
  - Display all employee records.
- **Implementation**: Uses Python dictionaries for efficient data storage and retrieval.
