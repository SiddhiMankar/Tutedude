# Python Basics Assignments

This directory contains fundamental Python programming exercises and assignments.

## Assignments

### Assignment 1: Basics
- **Task 1**: Arithmetic operations (+, -, *, /) between two numbers.
- **Task 2**: Personalized greeting using user input for first and last name.

### Assignment 2: Control Flow
- **Task 1**: Even or Odd number checker.
- **Task 2**: Calculate the sum of all numbers from 1 to 50 using a `for` loop.

### Assignment 3: Functions and Modules
- **Task 1**: Factorial calculation using recursion.
- **Task 2**: Using the `math` module for square root, sine, and natural logarithm calculations.

### Miscellaneous
- **`inlecture.py`**: Demonstrates list to tuple conversion and finding the minimum value in a collection.
